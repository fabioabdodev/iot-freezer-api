# Proposta Visual - Site Institucional Virtuagil

## Direcao geral

O site deve parecer:

- moderno
- confiavel
- comercial
- tecnologico sem ficar frio demais

## Sensacao desejada

Misturar:

- clareza de operacao
- sofisticacao leve
- proximidade comercial

## Direcao de design

### Tipografia

- titulos fortes e limpos
- corpo com leitura simples
- evitar cara de painel tecnico

### Cores

- base neutra forte
- acento tecnico controlado
- contraste bom para transmitir confianca

### Layout

- secoes bem definidas
- ritmo visual claro
- combinacao de conteudo comercial e imagem conceitual

### Elementos visuais

- cards para modulos e pacotes
- blocos explicativos
- CTAs evidentes
- icones objetivos

## Regra para a plataforma do cliente

A experiencia do cliente final deve seguir outra linha visual, separada do site institucional.

Ela deve parecer:

- elegante
- calma
- objetiva
- facil de entender em poucos segundos

O cliente final deve abrir e bater o olho em:

- status atual
- temperatura atual
- dentro ou fora da faixa
- ultimo envio
- alertas recentes

## Regra para a plataforma administrativa

A experiencia administrativa pode ser mais tecnica, com:

- tabelas
- filtros
- configuracoes
- diagnostico
- historicos mais profundos

## O que evitar

- visual generico de landing page barata
- cara de dashboard operacional no site institucional
- mesma interface para cliente final e administrador
- excesso de efeitos
- linguagem visual muito focada em developer

## Referencia conceitual

A Virtuagil deve parecer uma empresa que:

- entende operacao
- sabe automatizar
- sabe vender solucao real
- entrega leitura clara para o cliente e profundidade para a operacao interna
