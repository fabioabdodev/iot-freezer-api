# Precificacao Inicial - Rascunho

## Objetivo

Criar uma referencia simples de preco para nao chegar na conversa sem base.

No inicio, o mais importante e:

- nao cobrar abaixo do razoavel
- nao complicar demais a oferta
- validar se o cliente enxerga valor

## Modelo comercial recomendado no inicio

### 1. Implantacao

Valor unico para:

- preparacao do ambiente
- configuracao inicial
- instalacao basica
- ajuste de limites
- validacao inicial

### 2. Mensalidade

Valor recorrente para:

- monitoramento
- painel
- alertas
- acompanhamento basico
- manutencao operacional do servico

### 3. Adicional por equipamento

Quando passar do pacote base, cobrar por equipamento adicional.

## Estrutura simples de oferta

### Opcao A - Piloto enxuto

- 1 equipamento
- prazo de validacao
- implantacao reduzida
- mensalidade promocional ou validada

Uso:

- primeiro cliente
- prova de conceito

### Opcao B - Plano base

- 1 a 2 equipamentos
- implantacao normal
- mensalidade padrao

### Opcao C - Equipamentos extras

- valor adicional por freezer, geladeira ou ponto monitorado

## Como pensar o preco

Some estes fatores:

- custo do hardware
- seu tempo de implantacao
- custo mensal de infraestrutura
- suporte que voce vai prestar
- margem minima para o servico fazer sentido

## Formula simples

### Implantacao

`custo de hardware + tempo de implantacao + margem`

### Mensalidade

`infraestrutura + suporte + acompanhamento + margem`

## Faixas para discutir com calma

Como rascunho interno, voce pode pensar em tres niveis:

- entrada
- base
- ampliado

Sem travar em numero agora. O valor final deve considerar:

- tamanho do cliente
- quantidade de equipamentos
- urgencia
- valor percebido

## Regra importante

No comeco, nao tente otimizar preco perfeito.

Tente validar:

- se o cliente entende o valor
- se aceita piloto
- se aceita mensalidade

## Estrategia recomendada

Para o restaurante do seu amigo:

- oferecer piloto simples
- escopo pequeno
- valor claro
- prazo definido

Depois do piloto:

- ajustar preco com base em uso real, suporte e resultado percebido
