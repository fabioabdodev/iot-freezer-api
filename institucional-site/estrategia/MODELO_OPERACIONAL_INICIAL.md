# Modelo Operacional Inicial

## Objetivo

Definir como a Virtuagil deve operar na fase inicial do produto, sem complicar
demais a implantacao e sem exigir estrutura grande antes da hora.

## Regra principal da fase 1

Na fase inicial:

- a Virtuagil administra o sistema
- o cliente acessa via web para visualizar os dados dele

## Papel da Virtuagil

A Virtuagil fica responsavel por:

- cadastrar o cliente
- cadastrar os equipamentos
- configurar regras e limites
- implantar a solucao
- ajustar o monitoramento
- acompanhar suporte inicial

## Papel do cliente

O cliente, na fase inicial, deve:

- acessar o sistema via navegador
- visualizar apenas os dados da propria operacao
- acompanhar status, historico e alertas

No comeco, o cliente nao precisa administrar configuracoes criticas.

## Canal de acesso

O acesso do cliente deve acontecer por:

- web responsiva

Isso significa:

- uso no computador
- uso no celular via navegador
- sem app nativo nesta fase

## Motivo dessa escolha

Esse modelo e melhor agora porque:

- simplifica a implantacao
- reduz erro de configuracao
- permite mais controle da operacao
- facilita a venda inicial
- evita complexidade desnecessaria

## Estrutura de acesso desejada no futuro

### Area da Virtuagil

- visao administrativa global
- acesso a todos os clientes
- cadastro e configuracao completa

### Area do cliente

- login proprio
- visao restrita ao `clientId`
- acompanhamento do dashboard
- historico e alertas

## O que entra na fase 1

- voce controla toda a parte administrativa
- cliente acessa via web
- cliente enxerga apenas os dados dele
- foco em acompanhamento e valor operacional

## O que fica para depois

- app mobile nativo
- autonomia completa do cliente para configurar tudo
- niveis complexos de permissao
- painel administrativo multiusuario completo

## Regra de produto

Antes de expandir autonomia do cliente, a prioridade e:

1. validar uso real
2. validar proposta comercial
3. validar implantacao
4. validar valor percebido

So depois disso faz sentido abrir mais camadas administrativas.
