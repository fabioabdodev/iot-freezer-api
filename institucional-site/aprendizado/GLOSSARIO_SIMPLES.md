# Glossario Simples

## IoT

IoT significa "internet das coisas". No seu contexto, e quando um equipamento
ou sensor envia informacoes pela internet para um sistema.

## Sensor

E o componente que capta uma informacao do ambiente.

Exemplo:

- sensor de temperatura

## Device

E o equipamento ou dispositivo que envia os dados para o sistema.

Exemplo:

- um ESP32 com sensor conectado

## Dashboard

E a tela onde voce acompanha status, historico e alertas.

## Alerta

E o aviso disparado quando algo sai do normal.

Exemplo:

- temperatura fora da faixa
- equipamento offline

## Offline

Significa que o equipamento parou de reportar dados por um tempo maior que o
esperado.

## Modulo

Na Virtuagil, modulo e um bloco de solucao.

Exemplo:

- modulo temperatura
- modulo bots e automacoes WhatsApp

## n8n

Ferramenta de automacao de fluxos.

Serve para criar automacoes entre sistemas, notificacoes, atendimento e outros
processos.

## Bot de WhatsApp

Automacao que responde, organiza ou conduz parte do atendimento pelo WhatsApp.

## Piloto

Projeto inicial, pequeno e controlado, usado para validar se a solucao faz
sentido para o cliente.
