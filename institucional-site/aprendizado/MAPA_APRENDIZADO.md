# Mapa de Aprendizado

## Objetivo

Te ajudar a evoluir sem tentar entender tudo de uma vez.

## Etapa 1 - Entender o problema de negocio

Primeiro, foque em entender:

- que problema o cliente tem
- quanto custa esse problema
- como a Virtuagil pode resolver de forma simples

Voce nao precisa dominar hardware primeiro para vender uma ideia de valor.

## Etapa 2 - Entender o modulo temperatura

Depois, foque em:

- o que o modulo temperatura faz
- quando ele gera alerta
- o que o dashboard mostra
- como um piloto pode ser vendido

## Etapa 3 - Entender o fluxo do sistema

Mapa simples:

1. sensor/device envia dado
2. API recebe
3. banco salva
4. monitor analisa
5. alerta pode ser enviado
6. dashboard mostra o estado

## Etapa 4 - Entender automacoes com n8n

Aprender:

- o que e um fluxo
- o que pode ser automatizado
- como WhatsApp entra nisso

## Etapa 5 - Entender operacao

Quando se sentir mais seguro:

- como funciona deploy
- como ver logs
- como fazer teste com simulador
- como apresentar isso para cliente

## Regra importante

Voce nao precisa dominar:

- todo hardware
- toda infraestrutura
- toda automacao

para comecar.

Voce precisa entender primeiro:

- problema
- solucao
- valor
- proximo passo

## Ordem recomendada para voce

1. estudar a proposta comercial
2. entender o modulo temperatura
3. entender o dashboard
4. entender o simulador
5. depois aprofundar em IoT e automacao
