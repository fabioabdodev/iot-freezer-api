# Jade Knowledge Base (Copy/Paste para Google Docs)

Esta pasta centraliza os textos oficiais da assistente Jade para voce copiar e colar nos arquivos do Google Docs.

## Arquivos

- `JADE_SOBRE_EMPRESA.md`
- `JADE_PRODUTOS_E_PLANOS.md`
- `JADE_PRECOS_E_PROMOCOES.md`
- `JADE_FAQ_COMERCIAL.md`
- `JADE_REGRAS_ATENDIMENTO.md`

## Fluxo operacional recomendado

1. Atualize os arquivos desta pasta sempre que houver mudanca relevante.
2. Copie o conteudo atualizado e cole no Google Docs correspondente.
3. Atualize os campos de controle no topo (`versao`, `ultima_atualizacao`, `responsavel`).
4. Se houver mudanca comercial sensivel (preco, promocao, condicao), atualize primeiro `JADE_PRECOS_E_PROMOCOES.md`.

## Importante

- Nao inventar valores comerciais.
- Se faltar informacao em docs, a Jade deve encaminhar para suporte humano.
- Manter linguagem objetiva e comercial.
